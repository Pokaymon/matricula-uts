¡Bienvenido, Coordinador!

Desde aquí puedes gestionar los pensums, materias y asignar profesores según tu programa.

¿Qué puedes hacer como Coordinador?

  • Gestión de Pensums: Crea, edita o elimina pensums académicos para programas y carreras. Puedes actualizarlos según las necesidades curriculares.
  • Gestión de Materias: Crea, modifica o elimina materias dentro de los pensums. Define requisitos, créditos y otros detalles importantes.
  • Asignación de Profesores: Asigna docentes a las materias correspondientes dentro de cada pensum.

Ejemplos de tareas que puedes realizar

  • Crear un nuevo pensum para un programa académico.
  • Añadir nuevas materias a un pensum existente.
  • Asignar un profesor responsable para cada materia.
