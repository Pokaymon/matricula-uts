¡Bienvenido, Administrador!

Desde aquí puedes gestionar usuarios, asignar permisos y supervisar el sistema académico.

¿Qué puedes hacer como Administrador?

  • Gestión de usuarios: Crea, consulta, edita o elimina usuarios. Puedes asignarles roles específicos y permisos personalizados.
  • Administración de roles y permisos: Controla las acciones permitidas para cada tipo de usuario según su perfil.
  • Estadísticas y reportes: Accede a información clave sobre cursos, usuarios y el estado general del sistema.

Ejemplos de tareas que puedes realizar

  • Registrar nuevos Coordinadores Académicos.
  • Asignar permisos para la gestión de pensums y materias.
  • Atender y validar solicitudes de modificación de datos de usuarios.
